# Rechnungsvorlage

Freiberufler/Selbständige können mit der Vorlage eine HTML/PDF Rechnung erstellen.

* http://codepen.io/nikolayhg/pen/oXegzM

Full-page demo: http://codepen.io/nikolayhg/full/oXegzM/

## Features

* Data as JS objects
* Livereload of data
* Print to PDF
