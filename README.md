# Rechnungsvorlage

Freiberufler/Selbständige können mit der Vorlage eine HTML/PDF Rechnung erstellen.

* http://codepen.io/nikolaygit/pen/oXegzM

Full-page demo: http://s.codepen.io/nikolaygit/debug/oXegzM

* Print to PDF
